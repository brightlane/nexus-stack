# Nexus Stack Operations

**Status:** Deployment Active (Vulture 10K Protocol)
**Base URL:** https://brightlane.github.io/nexus-stack/

## Overview
Nexus Stack is a high-scale pSEO orchestration engine designed to automate the deployment of niche-specific business infrastructure. This repository manages the data-driven generation of thousands of landing pages across four core pillars:

1. **Nexus Logistics:** Supply chain and warehouse infrastructure.
2. **Nexus Finance:** Tax compliance and financial automation.
3. **Nexus Media:** Professional broadcast and creator tools.
4. **Nexus Growth:** B2B intelligence and conversion optimization.

## Structure
- `/data`: Contains `affiliate.json` (The partner source of truth).
- `/scripts`: Python and Node.js scripts for page generation.
- `/templates`: HTML/CSS components for the Chameleon engine.

## Automation
This repository uses **GitHub Actions** to refresh sitemaps and content weekly, ensuring all affiliate tracking links and offers are current for 2026.
